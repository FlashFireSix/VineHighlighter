# VineHighlighter
An accessibility tool that highlights the product category counts in Vine to show if they've increased or decreased on refreshing the page.

A toggle switch is shown next to the Vine logo to enable/disable it.

This is not signed by Mozilla as it was an experiment in agentic programming, use at your own risk.

To test, nagivate to about:debugging > This Firefox and under Temporary Extensions click Load Temporary Add-on.. then pick any of the files in the folder.
